# Почта (Post)

## Назначение
Здесь хранятся все письма, которые игрок получает в течение игры. Письма — основной двигатель сюжета. Через них транслируются задания (quests), моральные выборы и информация о мире.

## Структура файлов
- `day_XX.md` — письма конкретного игрового дня.
- `quests.md` — описание заданий, привязанных к письмам.

## Связь с игрой
Письма загружаются из таблицы `emails`. Поле `email_type` определяет иконку (почта/восклицательный знак/вопрос).
- `quest` — задание с выбором ответа.
- `info` — информационное письмо.
- `warning` — предупреждение системы.

## Статус
В процессе переноса из `scripts/database/game_content_firebird.sql`.
