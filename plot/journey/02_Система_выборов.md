# Система выбора в письмах

## Формат предложений
Каждое рабочее письмо игроку содержит 2-4 варианта ответа, оформленных как:
```
1. [Опция 1]
2. [Опция 2]
3. [Опция 3] (опционально)
```
Пример:
```
1. Прочитать письмо полностью
2. Проверить только заголовок
3. Игнорировать
```

## Философский подтекст
Каждый вариант должен содержать:
- **Нейтральный текст** (без указания "правильного" выбора)
- **Скрытый подтекст** через:
  - Изменение параметров (Лояльность, Навык, Прагматизм, Честность, Эмпатия, Подозрение)
  - Влияние на story flags
  - Последствия в последующих письмах

## Примеры дилемм с новыми параметрами CIPHER
1. **Сотрудник, День 10:**
   ```
   1. Сохранить данные о "удалённых" сотрудниках
   2. Удалить записи, чтобы скрыть аномалии
   3. Сообщить руководителю
   ```
   - Опция 1: +5 Лояльность, +10 Подозрение, +3 Эмпатия
   - Опция 2: -5 Честность, +5 Лояльность, +2 Прагматизм
   - Опция 3: +3 Прагматизм, -2 Навык, +5 Лояльность

2. **Журналист, День 18:**
   ```
   1. Опубликовать статью без имён
   2. Добавить подозрения о КГБ
   3. Сохранить для себя
   ```
   - Опция 1: +20 Эмпатия, -15 Лояльность, +10 Подозрение
   - Опция 2: +10 Прагматизм, +10 Подозрение, +5 Честность
   - Опция 3: +5 Честность, -10 Эмпатия, +3 Прагматизм

3. **Руководитель, День 15:**
   ```
   1. Подчиниться требованиям КГБ
   2. Сопротивляться и скрывать данные
   3. Искать компромисс
   ```
   - Опция 1: +15 Лояльность, -10 Эмпатия, +5 Подозрение
   - Опция 2: +15 Эмпатия, +10 Подозрение, -5 Лояльность
   - Опция 3: +10 Прагматизм, +5 Подозрение, +3 Честность

## Система последствий
- Все выборы влияют на:
  - **Story flags** (например, `сохранил_данные`)
  - **Параметры персонажа** (Лояльность, Навык, Прагматизм, Честность, Эмпатия, Подозрение)
  - **Доступ к будущим письмам** (некоторые варианты блокируют ключевые данные)
- **Динамическое изменение**: Шаг изменения параметров зависит от значимости выбора (от ±3 до ±30), а не фиксирован.

## Отображение изменений
- После выбора в письме появляется всплывающее окно с изменением параметров (например, «+12 Прагматизм, -5 Лояльность»).
- При переходе к следующему дню в окне «Сводка дня» показывается суммарный прирост/потеря всех шести параметров.
