# Концовки и story flags

## Основные story flags
- `открыл_протокол_феникс` – Роль открыла ключевые данные проекта
- `нашел_жертв_экспериментов` – Обнаружены 47 архивированных сотрудников
- `раскрыл_вовлеченность_кгб` – Выявлена роль КГБ в проекте
- `сохранил_данные` – Данные о жертвах сохранены (не удалены)
- `уничтожил_доказательства` – Все следы уничтожены
- `продал_информацию` – Информация продана третьей стороне
- `стал_информатором` – Персонаж стал информатором КГБ
- `опубликовал_статью` – Статья опубликована в электронной газете
- `спас_информатора` – Удалось спасти исходного информатора

## Условия концовок (примеры, использующие параметры CIPHER)
### Архивариус
- `сохранил_данные` = true
- Нарушения < 3
- Лояльность > 50   *(высокая преданность системе, но данные сохранены)*

### Разоблачитель
- `раскрыл_заговор` = true   *(флаг, активируемый при раскрытии вовлечённости КГБ и т.д.)*
- Подозрение > 70
- Честность > 80

### Призрак
- `исчез_из_системы` = true
- Подозрение > 90
- Навык > 80

### Новый директор
- `стал_директором` = true
- Прагматизм > 70   *(ресурсность и умение использовать ситуацию)*
- Лояльность > 60   *(доверие со стороны КГБ/системы)*

### Файербёрд восстает
- `открыл_сознание_иИ` = true
- Навык > 90
- Подозрение > 80

## Цепная реакция
- `раскрыл_заговор` → все роли получают +10 Подозрение
- `опубликовал_статью` → руководитель теряет -15 Честность (репутация падает)
- `саботировал_проект` → сотрудник получает меньше заданий (из-за снижения доверия)
