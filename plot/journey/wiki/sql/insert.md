# INSERT - добавление данных
**Категория:** sql  
**Постоянная:** да  
**Уровень:** начальный

```
Команда INSERT добавляет новые строки в таблицу.

Синтаксис:
INSERT INTO table_name (column1, column2) VALUES (value1, value2);

Примеры:
- INSERT INTO employees (id, name, department) VALUES (6, 'Новый Сотрудник', 'IT');
- INSERT INTO news_articles (title, content) VALUES ('Заголовок', 'Текст статьи');
