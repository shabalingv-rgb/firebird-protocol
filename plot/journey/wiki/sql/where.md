# WHERE - фильтрация данных
**Категория:** sql  
**Постоянная:** да

```
Ключевое слово WHERE фильтрует записи по условию.

Операторы сравнения:
= равно
<> не равно
> больше
< меньше
>= больше или равно
<= меньше или равно

Логические операторы:
AND — оба условия истинны
OR — хотя бы одно истинно
NOT — инверсия условия
