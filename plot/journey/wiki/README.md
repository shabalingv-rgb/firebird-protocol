# Wiki (Справочник)

## Назначение
Постоянные статьи, хранящиеся в базе. Справочные материалы: основы SQL, описание таблиц, истории института.

## Структура файлов
- `sql_basics.md` — основы SQL.
- `tables.md` — описание таблиц базы данных.
- `lore.md` — лор об институте и истории.

## Связь с игрой
Загружаются из `news_articles` с `category = 'wiki'` и `guide_topics`.

## Статус
В процессе переноса из `scripts/database/game_content_firebird.sql`.
