# Wiki: Таблицы базы данных

## Таблица: employees
**Категория:** tables  
**Постоянная:** да

```
Сотрудники НИИ "Файербёрд".

Структура:
- id — уникальный номер
- name — ФИО сотрудника
- department — отдел (IT, Accounting, Security, etc.)
- salary — зарплата в рублях

Примеры использования:
- Поиск сотрудников по отделу
- Анализ зарплат
- Подсчёт количества персонала
```

## Таблица: emails
**Категория:** tables  
**Постоянная:** да

```
Архив входящей почты.

Структура:
- id — уникальный номер письма
- day_id — игровой день
- sender — отправитель
- subject — тема
- body — содержимое
- email_type — тип (quest, info, warning)

Используется для отслеживания переписки.
```

## Таблица: quests
**Категория:** tables  
**Постоянная:** да

```
Задания для сотрудников.

Структура:
- id — номер задания
- email_id — связь с письмом
- title — название
- description — описание
- sql_template — ожидаемый SQL
- expected_rows — количество строк в результате
```

## Таблица: news_articles
**Категория:** tables  
**Постоянная:** да

```
Новости внутренней сети.

Структура:
- id — номер статьи
- day_id — когда опубликовано
- category — раздел (news, wiki, board, humor)
- title — заголовок
- content — текст
- author — автор
- is_permanent — постоянная статья (wiki)
```
