# Юмор (Humor)

## Назначение
Шуточные статьи для разрядки и создания атмосферы. Не влияют на сюжет, но добавляют характер.

## Структура файлов
- `day_XX.md` — шутки за конкретный день.

## Связь с игрой
Загружаются из `news_articles` с `category = 'humor'`.

## Статус
В процессе переноса из `scripts/database/game_content_firebird.sql`.
