# Новости (News)

## Назначение
Новостные статьи, доступные через Браузер. Передают атмосферу института, события дня, и могут содержать намёки на основной сюжет.

## Структура файлов
- `day_XX.md` — новости конкретного дня.
- `category_news.md` — общие новости.

## Связь с игрой
Загружаются из `news_articles` с `category = 'news'`. Используются как фоновый контент.

## Статус
В процессе переноса из `scripts/database/game_content_firebird.sql`.
