# Доска объявлений (Board)

## Назначение
Бытовые объявления сотрудников института. Создают ощущение живого сообщества, мелкие детали для атмосферы.

## Структура файлов
- `day_XX.md` — объявления за конкретный день.

## Связь с игрой
Загружаются из `news_articles` с `category = 'board'`.

## Статус
В процессе переноса из `scripts/database/game_content_firebird.sql`.
