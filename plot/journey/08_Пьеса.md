# ПРИЛОЖЕНИЕ: Система CIPHER

| Параметр | Описание | Влияние на финал |
|----------|----------|-----------------|
| **Лояльность** | Верность системе и институту | Архивариус, Новый директор |
| **Навык** | Владение SQL и аналитика | Призрак, Файербёрд |
| **Прагматизм** | Готовность к компромиссам | Новый директор |
| **Честность** | Стремление к правде | Разоблачитель |
| **Эмпатия** | Сочувствие жертвам | Архивариус |
| **Подозрение** | Внимание системы к вам | Призрак, Разоблачитель |
