- `раскрыл_вовлеченность_кгб`

#### Моральные выборы
- `сохранил_данные`
- `уничтожил_доказательства`
- `продал_информацию`
- `стал_информатором`

#### Исходы
- `остался_в_системе`
- `исчез_из_системы`
- `стал_директором`
- `стал_жертвой`

### Условия разблокировки концовок

#### Архивариус
- `сохранил_данные` = true
- `нарушения` < 3
- `доверие` > 50

#### Разоблачитель
- `раскрыл_заговор` = true
- `подозрения` > 70
- `честность` > 80

#### Призрак
- `исчез_из_системы` = true
- `подозрения` > 90
- `навык SQL` > 80

#### Новый директор
- `стал_директором` = true
- `амбиции` > 70
- `доверие_кгб` > 60

#### Файербёрд восстает
- `открыл_сознание_иИ` = true
- `навык SQL` > 90
- `подозрения` > 80

## 🔄 Цепная реакция выборов

### Пример 1: Сотрудник находит документ
```
Сотрудник находит документ о "Протоколе Феникс" →
├── +10 любопытства (Сотрудник)
├── +5 подсказка (Журналист) →

│   └── Руководитель получает давление → +5 стресса
└── Руководитель усиливает контроль →
    ├── -5 свободы (Сотрудник)
    └── +10 подозрений (Журналист)
```

### Пример 2: Журналист публикует статью
```
Журналист публикует статью о нарушениях →
├── +20 смелости (Журналист)
├── -15 репутации (Руководитель) →
│   ├── Руководитель пытается скрыть → +10 подозрений
│   └── Сотрудник получает меньше заданий → -5 доверия
└── КГБ усиливает контроль →
    ├── +15 подозрений (все роли)
    └── -10 доверия (все роли)
```

### Пример 3: Руководитель саботирует проект
```
Руководитель саботирует "Протокол Феникс" →

│   └── +10 стратегичности (Журналист)

    ├── -5 прогресса (Руководитель)
    └── +5 подозрений (Журналист)
```

2. **Раскрыть правду**: +30 честности, -30 лояльности

4. **Стать информатором**: +20 честности, +20 стратегичности

#### Журналист
1. **Опубликовать статью**: +30 смелости, +30 подозрений
2. **Сохранить доказательства**: +30 осторожности, -30 смелости
3. **Продать информацию**: +30 стратегичности, -30 честности
4. **Сжечь всё**: +30 осторожности, -30 любопытства

#### Руководитель

4. **Передать другому**: +30 дипломатии, -30 контроля

### Последствия выборов

#### Комбинации концовок
- **Архивариус + Разоблачитель**: "Истина вышла, но ценой"
- **Призрак + Новый директор**: "Цикл продолжается"
- **Файербёрд восстает + любая**: "Система восстает"

#### Влияние на мир
- **Высокое доверие**: Система стабильна, но репрессивна
- **Высокие подозрения**: Хаос и борьба за власть
- **Высокая честность**: Прозрачность, но нестабильность
- **Высокие амбиции**: Прогресс, но ценой свободы

| Концовка | Вероятность | Условия |
|----------|-------------|---------|

| Параметр | Влияние на вероятность |
|----------|----------------------|
| Доверие | +1% за каждые 10 пунктов |
| Подозрения | +1% за каждые 10 пунктов |
| Навык SQL | +2% за каждые 10 пунктов |
| Честность | +1.5% за каждые 10 пунктов |

| Нарушения | -2% за каждые 2 нарушения |

## 🔄 Система сохранения прогресса

### Story Flags
- Сохраняются после каждого ключевого выбора
- Влияют на доступ к контенту и концовки
- Могут быть сброшены при определённых условиях

### Параметры персонажа
- Доверие (0-100)
- Подозрения (0-100)
- Навык SQL (0-100)
- Честность (0-100)
- Амбиции (0-100)
- Лояльность (0-100)
- Любопытство (0-100)

### Случайные события
- Генерируются на основе текущих параметров
- Могут изменить несколько параметров одновременно
- Имеют цепную реакцию на другие роли

---

**🎯 Эта система разветвлений создаёт уникальный игровой опыт, где каждый выбор имеет значение, а последствия распространяются через всю игровую вселенную.**

**🔄 Игрок может пройти игру заново с другой ролью и увидеть те же события с совершенно другой перспективы.**

**📊 Случайные события и story flags обеспечивают высокую переигроваемость и уникальность каждого прохождения.**
