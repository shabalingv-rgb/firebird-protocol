# План исправления ошибок CS0103 в FirebirdDatabase.cs

## Описание проблемы
В файле `scripts/system/FirebirdDatabase.cs` вызываются два метода, которые не определены:
- `MigrateQuestsTable()` (строка 410)
- `EnsureTutorialQuest()` (строка 413)

Это приводит к ошибкам компиляции CS0103: "Имя не существует в текущем контексте".

## Решение
Необходимо добавить недостающие методы в класс `FirebirdDatabase`.

### Метод MigrateQuestsTable
Этот метод должен проверять наличие столбца `procedure_type` в таблице `quests` и добавлять его, если он отсутствует.

### Метод EnsureTutorialQuest
Этот метод должен проверять наличие квеста с `email_id = 2` (квест-инструктаж) и добавлять его, если он отсутствует.

## Шаги реализации
1. Открыть файл `scripts/system/FirebirdDatabase.cs`.
2. Найти строку 445 (после метода `LoadContentToCache` и перед методом `ExecuteQuery`).
3. Вставить определения двух методов после закрывающей фигурной скобки метода `LoadContentToCache` и перед комментарием `/// <summary>Выполнить SELECT из GDScript (терминал)...`.
4. Убедиться, что методы используют существующие helper-методы класса: `ExecuteQuery` и `ExecuteNonQuery`.
5. Сохранить файл.
6. Проверить компиляцию.

## Ожидаемый результат
После добавления методов ошибки CS0103 должны исчезнуть, и проект должен компилироваться успешно.
